# Design and Verification of a Performance-Enhanced Hazard Detection and Resolution Unit in a Pipelined Processor

## Project Overview
This project implements a 5-stage pipelined processor integrated with an enhanced Hazard Detection and Resolution Unit (HDU) and Built-In Self-Test (BIST) architecture.

## Features
- 5-stage pipeline processor
- Hazard Detection Unit
- Forwarding Unit
- Pipeline stall mechanism
- Branch handling
- Built-In Self-Test (BIST)
- LFSR-based test pattern generation
- MISR-based response analysis
- Verilog HDL implementation
- Functional verification

## Tools Used
- Verilog HDL
- Cadence/Xcelium
- GTKWave/SimVision

## Project Flow
Single Cycle Processor
→ Pipelined Processor
→ Hazard Detection Unit
→ Forwarding Logic
→ BIST Integration
→ Verification and Analysis
